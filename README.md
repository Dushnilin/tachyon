# Tachyon

[![Star](https://img.shields.io/github/stars/Dushnilin/tachyon?style=social)](https://github.com/Dushnilin/tachyon/stargazers)
[![Releases](https://img.shields.io/github/v/release/Dushnilin/tachyon?label=releases)](https://github.com/Dushnilin/tachyon/releases)

> **Tachyon** — это форк проекта **Forkop** (ранее известного как **Podkop Plus**). Проект переименован и развивается как независимое решение для OpenWrt на базе оригинального [Podkop](https://github.com/itdoginfo/podkop).

Tachyon предоставляет мощный инструментарий для гибкой маршрутизации сетевого трафика (через прокси-серверы, средства обхода блокировок или напрямую) на основе доменов, IP-адресов и портов с помощью связки **sing-box**, **zapret** и **nftables**. Вся логика бэкенда написана на легковесном скриптовом языке **ucode** (родном для OpenWrt).

---

### Установка

Выполните команду в терминале вашего роутера:

```sh
sh <(wget -O - https://raw.githubusercontent.com/Dushnilin/tachyon/main/install.sh)
```

> [!NOTE]
> Конфигурация от **Forkop** (или **Podkop Plus / Podkop**) полностью совместима. Во время установки скрипт автоматически перенесёт все ваши настройки в новый конфигурационный файл `/etc/config/tachyon`!

### Что нового в Tachyon: сравнение с оригинальным Podkop и Forkop

Tachyon значительно расширяет возможности классического Podkop, делая управление обходом блокировок автоматизированным, отказоустойчивым и удобным.

#### 1. Интерактивный Telegram-бот
В отличие от базового мониторинга в старых версиях, бот в Tachyon превратился в полноценный пульт управления роутером:
* **Управление списками «на лету» (`/rules`)**: Бот выведет список всех правил обхода. Вы можете выбрать любую секцию и прямо из чата добавлять в неё новые домены/IP (в том числе списком из нескольких строк) или удалять их. Бот сам применит изменения в UCI и перезапустит правила.
* **Быстрое включение/отключение правил**: Любую секцию маршрутизации можно мгновенно активировать или деактивировать кнопкой в боте.
* **Переключение серверов (`/servers`)**: Интерактивный выбор активного прокси-сервера (с отображением текущих задержек).
* **Контроль устройств (`/devices`)**: Просмотр активных DHCP-клиентов с возможностью заблокировать или разблокировать интернет любому устройству по MAC-адресу в один клик.
* **Диагностика системы (`/doctor`)**: Запуск проверки здоровья сети и исправление ошибок прямо из мессенджера.

#### 2. Инструмент автоматического ремонта — Tachyon Doctor
В систему встроена CLI-утилита проверки работоспособности, доступная по команде `tachyon doctor`:
* Сканирует статус процессов (`sing-box`, `dnsmasq`, `watchdog`).
* Проверяет таблицы `nftables` и корректность прохождения DNS-запросов через localhost.
* Тестирует доступность Clash API.
* При обнаружении проблем (упавшая служба, сбившиеся правила файрвола, ошибки резолвера) утилита **автоматически исправляет их** без перезагрузки всего роутера.

#### 3. Горячая перезагрузка (Hot-reload)
* При обновлении подписок, переключении серверов или изменении правил обхода Tachyon больше не перезапускает службу `sing-box` полностью.
* Вместо этого используется сигнал `SIGHUP` для мягкого обновления конфигурации на лету. Сетевые соединения клиентов не разрываются, а интернет не пропадает во время перезагрузки.

#### 4. Watchdog и защита от нехватки памяти (OOM Audit)
* Добавлен фоновый процесс контроля стабильности (Watchdog), следящий за работой sing-box.
* Интегрирован аудит оперативной памяти: в случае OOM (Out of Memory) событий система автоматически пришлет оповещение в Telegram и оптимизирует лимиты памяти, подстраиваясь под слабые роутеры.

#### 5. Корректный MTU Fix (MSS Clamping)
* Добавлен автоматический обход проблем с зависанием TCP-соединений на VPN-интерфейсах.
* Внедрена автонастройка MSS Clamping в `nftables` с автоматическим переключением на фиксированный размер пакета (`1400`), если ядро роутера не поддерживает динамический расчет (`rtmtu`).

#### 6. Поддержка новых протоколов и инструментов обхода
* Полноценная поддержка ядра `sing-box extended` и современного транспорта `XHTTP`.
* Встроенная интеграция утилит обхода DPI: **Zapret**, **Zapret2** и **ByeDPI** могут быть назначены как индивидуальные действия (actions) для каждого правила маршрутизации отдельно.
* Поддержка каскадного проксирования (построение цепочек из нескольких прокси) и групп тестирования задержек (`URLTest`) для автоматического выбора самого быстрого сервера.

#### 7. Современный веб-интерфейс (LuCI)
* Фронтенд полностью переписан на TypeScript (`fe-app-tachyon`) и собирается в отимизированный бандл.
* Полная совместимость со всеми темами оформления OpenWrt (включая Argon, Bootstrap, Material). Интерфейс адаптирован под мобильные устройства и имеет модульную разбивку настроек.
