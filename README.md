# Tachyon

[![Star](https://img.shields.io/github/stars/Dushnilin/tachyon?style=social)](https://github.com/Dushnilin/tachyon/stargazers)
[![Releases](https://img.shields.io/github/v/release/Dushnilin/tachyon?label=releases)](https://github.com/Dushnilin/tachyon/releases)

> **Tachyon** — это форк проекта **Forkop** (ранее известного как **Podkop Plus**). Проект переименован и развивается как независимое, современное решение для OpenWrt. Мы сохранили всё лучшее из оригинального [Podkop](https://github.com/itdoginfo/podkop), из которого вырос наш код, и превратили его в мощный, надежный и удобный "комбайн" для управления сетью.

Tachyon предоставляет мощный инструментарий для гибкой маршрутизации сетевого трафика (через прокси-серверы, средства обхода блокировок или напрямую) на основе доменов, IP-адресов и портов. Вся логика бэкенда написана на легковесном скриптовом языке **ucode**, который является родным для OpenWrt.

---

### Что умеет Tachyon?

Tachyon значительно расширяет возможности классических решений и делает управление обходом блокировок автоматизированным, отказоустойчивым и очень удобным.

#### 🛡 Обход блокировок и проксирование (Мульти-протоколы)
* **sing-box**: Поддержка современных протоколов (VLESS, VMess, Shadowsocks, Trojan, WireGuard) для проксирования трафика.
* **Интеграция с Zapret (nfqws / nfqws2)** и **ByeDPI**: Встроенные средства для локального обхода DPI (глубокого анализа пакетов) без использования сторонних серверов.
* **Умная маршрутизация**: Настраивайте правила (домены, IP, порты), чтобы пускать через прокси или средства обхода только то, что нужно.
* **Поддержка подписок**: Автоматическое фоновое обновление прокси-серверов по вашим ссылкам.

#### 🤖 Интерактивный Telegram-бот
Бот в Tachyon — это полноценный пульт управления вашим роутером прямо из мессенджера:
* **Управление списками «на лету»**: Просматривайте правила обхода и добавляйте в них новые домены/IP (даже целым списком). Бот сам всё применит.
* **Быстрое переключение**: Включайте и отключайте любые секции маршрутизации одной кнопкой.
* **Переключение серверов**: Выбирайте активный прокси-сервер (бот покажет текущие задержки).
* **Контроль устройств**: Просматривайте активных DHCP-клиентов и блокируйте/разблокируйте им интернет (по MAC-адресу) в один клик.
* **Диагностика**: Запускайте проверку сети прямо из чата.

#### 🩺 Инструмент автоматического ремонта — Tachyon Doctor
В систему встроена мощная система диагностики (доступна через `tachyon doctor` или из Web-интерфейса):
* Сканирует статус служб (`sing-box`, `zapret`, `byedpi`, `dnsmasq`, `watchdog`).
* Проверяет правила `nftables` и корректность DNS-запросов через `localhost`.
* При обнаружении проблем утилита **автоматически исправляет их** (перезапускает службы, чинит правила файрвола) без полной перезагрузки роутера.

#### ⚡ Надежность, Hot-reload и Watchdog
* **Горячая перезагрузка (Hot-reload)**: При обновлении правил или переключении серверов Tachyon использует мягкое обновление конфигурации. Ваши текущие загрузки и звонки в интернете не прервутся!
* **Watchdog и защита от нехватки памяти**: Фоновый процесс постоянно следит за здоровьем системы. Если роутеру не хватает памяти (OOM), система оптимизирует лимиты и пришлет оповещение в Telegram.
* **Корректный MTU Fix (MSS Clamping)**: Автоматическое решение проблем с зависанием TCP-соединений на VPN-интерфейсах.

#### 🖥 Современный Web-интерфейс (LuCI)
* Полноценная интеграция в графический интерфейс OpenWrt.
* Удобный дашборд с мониторингом задержек, управлением подписками, компонентами системы и правилами в несколько кликов.

### Установка

Выполните команду в терминале вашего роутера:

```sh
sh <(wget -O - https://raw.githubusercontent.com/Dushnilin/tachyon/main/install.sh)
```

> [!NOTE]
> Конфигурация от **Forkop** (или **Podkop Plus / Podkop**) полностью совместима. Во время установки скрипт автоматически перенесёт все ваши настройки в новый конфигурационный файл `/etc/config/tachyon`!
